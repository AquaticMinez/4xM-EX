# 4xM-EX
Just a quick rundown, this modpack serves as an unofficial expansion pack that includes additional content and cosmetics for the original mod,4xM as the name also has inspiration from the brawl mod "Project Plus EX." This includes more skins, new params, and possibly in the future, New Characters!



Disclaimer: I do NOT own any assests from the Original 4xM mod,and ANY additional content added to 4xM EX will be credited properly by their creators.
Make SURE you don't use this mod with Vanilla 4xM as it WILL result in desyncs and possibly bans from Pretendo.


There's a lot of things I plan on for this project. As of May 3rd,2025, this mod is still in it's beta and has not been released publicly yet.
